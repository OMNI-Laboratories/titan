# Titan

A Go based Rest API for blockchain interactions

## Commands

run with:
`go run ./cmd/main.go`

test with:
`go test ./...`

## Third Party Services

### 0x

[0x](https://0x.org/docs/) provides swap rates and data that can be submitted to the blockchain.
